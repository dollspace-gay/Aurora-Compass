1//! User interface for Aurora Compass
2//!
3//! This crate provides the UI layer, including components,
4//! screens, navigation, theming, and design system primitives.
5//!
6//! # Design System
7//!
8//! The design system is built around Aurora borealis colors:
9//! - Primary: Aurora purple (#9D4EDD)
10//! - Secondary: Aurora cyan (#06FFA5)
11//! - Accent: Compass gold (#FFB703)
12//!
13//! Three themes are supported:
14//! - [`theme::ThemeName::Light`] - Bright theme with white background
15//! - [`theme::ThemeName::Dark`] - Dark theme with near-black background
16//! - [`theme::ThemeName::Dim`] - Softer dark theme
17//!
18//! # Modules
19//!
20//! - [`theme`] - Theme provider, color palettes, and gradients
21//! - [`tokens`] - Design tokens (spacing, sizing, breakpoints, etc.)
22//! - [`typography`] - Typography system and text styles
23//! - [`components`] - UI component library
24//! - [`screens`] - Application screens
25//! - [`navigation`] - Navigation framework
26//!
27//! # Example
28//!
29//! ```rust
30//! use app_ui::theme::{ThemeName, get_theme, ThemeState};
31//! use app_ui::tokens::spacing;
32//! use app_ui::typography::{Typography, TypographyVariant};
33//!
34//! // Get a theme
35//! let theme = get_theme(ThemeName::Dark);
36//! assert!(theme.is_dark());
37//!
38//! // Use design tokens
39//! let padding = spacing::SPACE_MD;
40//!
41//! // Get typography styles
42//! let typo = Typography::default();
43//! let title_style = typo.get(TypographyVariant::Title);
44//! ```
